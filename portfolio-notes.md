# 转盘生存 - 作品集说明

## 项目定位

《转盘生存》是一个基于“高频点击 + 受控选择攻击”框架的社交娱乐场原型。它不复刻真实老虎机，而是把转盘当作免费游戏的会话控制器：转次数控制操作节奏，金币推动基地进度，攻击/掠夺触发社交情绪，护盾缓冲挫败，限时活动提供短期目标。

## 核心循环图

```text
阶段开始
-> 获得免费转次数
-> 转动并获得奖励
-> 升级当前阶段建筑
-> 免费转次数耗尽
-> 1 美元购买阶段抽奖机会
-> 付费抽奖结算出补齐升级所需的金币
-> 升满 5 个建筑并进入下一阶段
```

## 系统结构图

```text
三阶段基地
  -> 每阶段 5 个新建筑
  -> 当前阶段全满级后进入下一阶段

转次数
  -> 转盘结果
      -> 金币
          -> 建筑升级
      -> 攻击
          -> 目标受损 + 金币 + 活动积分
      -> 掠夺
          -> 金库奖励 + 活动积分
      -> 护盾
          -> 抵挡来袭
      -> 大奖
          -> 大量金币 + 活动进度

活动积分
  -> 里程碑
  -> 排行榜名次

解锁规则
  -> 第二阶段解锁自动转动与 2 倍下注
  -> 第三阶段解锁 5 倍下注

阶段补给
  -> 免费转次数为 0 后点击转动
  -> 支付 1 美元
  -> 获得阶段抽奖机会
  -> 抽奖奖励合计刚好补齐当前阶段剩余建筑升级金币
```

| 每阶段建筑 | 5 个新建筑 |
| 第一阶段免费转次数 | 20 |
| 第二阶段免费转次数 | 24 |
| 第三阶段免费转次数 | 28 |
| 初始金币 | 0 |
| 初始护盾 | 1 |
| 倍率 | 1倍默认，2倍第二阶段解锁，5倍第三阶段解锁 |
| 自动转动 | 第二阶段解锁 |
| 建筑等级 | 每个 5 级 |

## 设计亮点

- 高频中奖：大多数转动都有金币收益，强调参与感而非真实赌博。
- 情绪转换：攻击/掠夺把随机结果转化为复仇、争夺、炫耀和损失厌恶。
- 浅层建造：5 个建筑槽提供进度感，不抢走单按钮核心。
- 限时活动：活动积分、里程碑和排行榜展示短局目标。
- 阶段漏斗：免费转次数一定先耗尽，随后通过 1 美元购买阶段抽奖机会；付费抽奖单次奖励随机波动，但总收益补齐当前阶段剩余升级金币。
- 阶段解锁：第二阶段解锁自动转动与 2 倍下注，第三阶段解锁 5 倍下注。

## 简历描述

设计并实现一款社交娱乐场原型，将转盘的高频随机奖励与三阶段基地升级、异步玩家对抗、攻击/掠夺、护盾、限时活动和排行榜结合。原型包含阶段解锁、免费转次数耗尽、1 美元购买阶段抽奖机会、付费抽奖结算补齐剩余升级金币等商业化节奏设计。

## 面试讲解词

我没有把转盘当成单纯的随机赔付机器，而是把它拆成免费游戏的会话控制器。每个阶段先给有限免费转次数，玩家会在升满当前阶段前耗尽机会；随后 1 美元购买阶段抽奖机会，金币仍然通过转动结算获得，而不是充值后直接到账。付费抽奖的单次金币奖励有随机波动，但整组抽奖的总收益会补齐当前阶段剩余升级金币。第二阶段开放自动转动和 2 倍下注，第三阶段开放 5 倍下注，用阶段推进承载功能解锁和付费节奏。

## 演示路径

1. 点击转动，展示金币收益和转盘动画。
2. 用金币升级一个基地建筑。
3. 继续转动触发护盾，展示基地保护。
4. 继续转动触发攻击，选择目标建筑并获得奖励。
5. 继续转动触发掠夺，选择藏点并获得大量金币。
6. 用完当前阶段免费转次数，再次点击转动打开 1 美元补给。
7. 领取阶段抽奖机会，继续转动并获得补齐升级所需的金币。
8. 升满当前阶段 5 个建筑并进入下一阶段。
9. 第二阶段展示自动转动和 2 倍下注解锁，第三阶段展示 5 倍下注解锁。
